# 狼人杀工具 v1.0

[![Python Version](https://img.shields.io/badge/python-3.7%2B-blue.svg)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![NumPy](https://img.shields.io/badge/NumPy-1.20%2B-orange.svg)](https://numpy.org/)

一款基于概率统计、蒙特卡洛模拟和离线语音转文字的狼人杀辅助分析工具。支持多算法融合分析、身份追踪、行为权重系统，以及 **faster-whisper 离线语音转文字**功能。

---

## ✨ 核心特性

### 🎲 多算法概率分析
- **蒙特卡洛模拟** - NumPy 加速，支持 50,000+ 次/秒模拟
- **三角定律** - 基于 85% 双狼定理的概率分析
- **贝叶斯更新** - 结合行为权重的动态概率更新
- **综合分析** - 融合多种算法的加权计算结果

### 🎴 智能身份系统
- 支持 50+ 种身份标记（狼人、神职、平民、好人标记）
- 实时身份状态追踪
- 好人标记自动分配（可分配到神或民）
- 三角形格局可视化分析

### ⚖️ 行为权重系统
- 内置 30+ 种预设行为标签
- 自定义标签权重（狼权/神权/民权）
- 贝叶斯动态更新概率

### 🎙️ 离线语音转文字（新增）
- 基于 **faster-whisper** 的离线语音识别
- 支持 GPU/CPU 自动检测
- 内置狼人杀术语热词（狼人、预言家、女巫、查杀、悍跳等）
- 录音文件自动保存，可直接转写填入发言记录

### 🎨 现代化交互界面
- 仿网易狼人杀号码牌设计
- 深浅色主题自由切换
- 点击卡片快速选择
- 完整的配置保存/加载功能

## 📦 安装指南

### 环境要求
- Python 3.7 或更高版本
- Windows / macOS / Linux

### 安装步骤

1. **克隆或下载项目**
   ```bash
   git clone https://github.com/haisi-ai/Werewolves.git
   cd Werewolves
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

   或手动安装：
   ```bash
   pip install numpy tkinter  # tkinter 通常已内置
   ```

3. **语音转文字功能（可选）**
   ```bash
   pip install faster-whisper pyaudio torch
   ```

   > 注意：PyAudio 在 Windows 上可能需要安装 `pip install pyaudio`，在 Linux 上可能需要 `sudo apt-get install portaudio19-dev`

4. **运行程序**
   ```bash
   python 狼人杀工具.py
   ```

## 🚀 快速开始

### 基础使用流程

1. **选择玩家** - 点击号码牌或从下拉框选择
2. **标记已知身份** - 在左侧面板选择玩家身份
3. **添加行为权重（可选）** - 根据玩家发言添加行为标签
4. **选择算法** - 蒙特卡洛/三角定律/贝叶斯/综合分析
5. **运行计算** - 查看概率结果，号码牌实时更新概率

### 语音转文字使用

1. 切换到 **"💬 发言记录"** 标签页
2. 选择玩家编号和轮次
3. 点击 **"⏺ 开始录音"** 开始录制
4. 录制完成后点击 **"⏹ 停止录音"**
5. 点击 **"📝 转写录音"** 等待识别完成
6. 识别结果自动填入发言文本框
7. 点击 **"💾 保存发言"** 保存记录

## 📊 算法说明

### 三角定律
12 人局中，4 个三角形（1-5-9、2-6-10、3-7-11、4-8-12）。  
理论概率：至少一组双狼的概率为 **83.64%**。

### 三行定律
连续三行（每行 2 人）至少出现 3 狼的概率约 **62.5%**。

### 四角定律
四角（每角 4 人）狼人分布概率分析。

### 贝叶斯更新
结合行为权重（狼权/神权/民权）动态更新先验概率。
