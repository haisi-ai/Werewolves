📖 简介
狼人杀工具是一款专业的狼人杀概率分析工具，通过多种算法融合分析，帮助玩家计算不同身份的概率分布。工具采用模块化设计，支持深色/浅色主题切换，提供直观的可视化界面。

✨ 主要特性
多算法融合分析

蒙特卡洛模拟（NumPy加速，50000+次/秒）

三角定律（85%双狼定理）

贝叶斯更新（权重系统）

综合分析（加权融合）

智能身份系统

支持 50+ 种身份标记

好人标记自动分配

自定义标签权重

实时身份状态追踪

可视化交互

实时概率更新

📱 使用指南
基本操作流程
选择玩家：点击号码牌或从下拉框选择玩家编号

标记身份：在身份输入页选择对应的身份

设置权重（可选）：在权重设置页添加行为权重

选择算法：在分析设置页选择要使用的算法

开始计算：点击对应的计算按钮查看结果

📊 数学原理
组合数学基础
总组合数：C(12,4) = 495

每组各1狼概率：81/495 = 16.36%

至少一组双狼概率：83.64%

三角定律
四个三角形分组：(1,5,9)、(2,6,10)、(3,7,11)、(4,8,12)

对角关系：159 ↔ 3711，2610 ↔ 4812

跳棋原则：双狼三角形对角大概率出狼

贝叶斯更新
基于先验概率和行为权重动态更新

支持多权重协同计算

自动归一化处理
